## Features

- ✨ Layout de landing page com seções de destaque e contato
- 🚀 CSS modular com variáveis, classes utilitárias e menu responsivo
- ⚡ Ilustrações SVG para background, JavaScript e contato
- 🎯 Boilerplate reutilizável para páginas de captura
- 🔧 Deploy publicado na Netlify
